Lesson 9: The Index

Indexing things like strings in python is a crucial skill. It allows us to access letters from certain positions in the sequence. Interestingly, the counting actually starts from 0 when it comes to indexing. So, the first letter has an index of 0, the second has an index of 1, and so on. Python is also extra nice and lets us index backwards! This, however, starts from -1, not 0. That means, the last letter has an index of -1, the second to last has an index of -2, and so on.

As mentioned in the previous exercise, indexing is done by using square brackets on a string. Try combining it with our other newfound knowledge below.

Task

Write a function everyOther that takes a single string s, which is guaranteed to be 6 characters long, and combines the 2nd, 4th and 6th indices and prints the result.

Sample Input 0

everyOther("davida")

Sample Output 0

aia
