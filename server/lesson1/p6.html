Lesson 6: Right and Wrong?

Booleans don't exist solely in the world of mathematical logic. Booleans also have their own special logical operators that apply only between themselves. You might know these better than you realize - they are and, or, and not. Some of these operators require two booleans, one on either side of them, and return just one that is the result of the operation. The and operator only returns True if both sides are true; the or operator returns True if either or both sides are true; the not operator, on the other hand does not require two booleans because all it does is invert the value of whatever follows. For example, the following:

not True and not False

can be simplified first into False and True, which overall is False because the and operator only returns True when both sides are true.

Since arithmetic comparisons also simplify into booleans, they can be used with boolean operators too. For example, the expression:

1 == 1 or 5 < 2

is True because the operator involved was or and at least one of the sides simplified into True.

Task

Write a function logic that accepts one boolean and two integers a, x, and y, and prints the boolean result of inverting a and using the and operator between the inverted a and whether or not x and y are equal to each other.

Sample Input 0

logic(False, 5, 5)

Sample Output 0

True
