Lesson 5: Right or Wrong

Yet another data type available to us in python is the boolean. Boolean simply means True or False. A lot of math operations result in boolean values. For example, 2 < 5 is True! To represent the "or equal to"s, simply follow the operator with an equals sign, <= and >=.

In python the = symbol is already used for assignment, which we'll go over in a future lesson. To determine if two things are equal to each other, we need the operator ==. Lastly, to see if things are not equal to each other, we use the operator !=.

Task

Write a function compare that accepts two numbers x and y, and prints whether x is less than, greater than, less than or equal to, greater than or equal to, equal to, and not equal to y, in that order.

Sample Input 0

compare(1, 2)

Sample Output 0

True
True
False
False
False
True
