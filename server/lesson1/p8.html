Lesson 8: Strings

We talked briefly before about strings, but let's get a little more in depth. A string in python is wrapped in either single or double quotes, and stores text as a 'string' of characters. Strings have operators and functions that we can employ to play with them. Adding two strings together with the + symbol concatenates the strings together, and multiplying a string with a number concatenates that string with itself as many times as the number provided. Finding the length of a string is done with the built-in python function len(). Finding what letter in a string is at a certain position is done with indexing like 'hello'[2], and finally whether a string is in another string can be determined by the in keyword in python. That last one especially is a huge help!

Task

Write a function concatIn that takes three strings x, y, and z, and prints whether or not the concatenation of x and y is in z.

Sample Input 0

concatIn("abc", "def", "qwerty")

Sample Output 0

False
