Lesson 1: Hello World

To make sure that you're set up properly, let's start with a simple Hello World script. Printing things in python can be achieved with the built in function print(), used like so:

print("Hello!")

You'll notice that we put our phrase in quotes, which tells python that we're working with a data type called a string. As we go on, we'll encounter more data types and their uses, but strings are a great place to start!

Task

Write a function that prints Hello, World!

Sample Output 0

Hello, World!
