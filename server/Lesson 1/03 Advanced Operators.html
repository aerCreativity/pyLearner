Lesson 3: Advanced Operators

Okay, so our four basic arithmetic operators are a walk in the park. A lot of the tasks we need to do in python require some more complicated techniques, however. Let's talk about some of the other operators available to us.

The division / operator that we used previously is great, but it turns values like 11/3 into big ugly numbers with lots of decimal points. Sometimes we only want to know how many times our number divided evenly, and the integer division // operator does exactly that. Now, 11//3 is a nice even 3. Notice that no rounding is taking place, python simply removed everything past the decimal point!

What if we wanted to know how much of the number didn't round evenly? The modulo % operator would be a great place to start. It gives us the remainder of any division operation. For example, 7%2 is 1. Mod 2 is very useful for determining if a number is even or odd!

The last extra operator we'll want to be using is the exponent ** operator. Many people are familiar with representing exponents with a ^, but that does something different in python. Exponents are useful for obvious reasons. Try it out with 2**5 to get 32.
