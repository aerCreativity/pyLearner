Lesson 2: Arithmetic

Let's get familiar with numbers and how to mess with them! Integers and floats are two more types of data that can be stored in python. In fact, to find out what data type something is, you can use the function type() like we did with print() earlier.

Integers represent whole numbers and floats represent fractions, although you'll find you can generally treat them interchangeably. We can also do operations on these number types, just like in math. Our most familiar ones, namely +, -, *, and /, are a great way to test that we're doing things right.

Task

Write a function operators that accepts two numbers x and y, and prints their sum, difference, product, and quotient, in that order.

Sample Input 0

operators(1, 2)

Sample Output 0

3
-1
2
0.5
