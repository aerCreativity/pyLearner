Midterm 1: Divergent

A word is divergent if it begins and ends with the same letter of the alphabet. For example, the word "area" is divergent, but the word "land" is not.

Task

Write a function named getDivergent that computes and returns how many words in a list of words are divergent and how many are not. The function getDivergent takes a single parameter, words, that is a list of non-empty, lowercase strings. The function getDivergent should print a pair of integers in the form of a tuple, the first being the number of divergent words in words and the second being the number of non-divergent words in words.

Sample Input 0

getDivergent(['apple', 'bob', 'cow', 'dad'])

Sample Output 0

(2, 2)
