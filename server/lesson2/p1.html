Lesson 1: Lists

A list is a neat way to store a collection of multiple objects. Lists in python can hold any data type. In fact, lists themselves are a data type, so lists can store other lists!

Lists are denoted with square brackets, like so:

['these', 'are', 6, 'elements', 'of', ['a', 'list', False]]

Lists share quite a lot of useful features with strings. For example, it's possible to see if something is a member of a list using the keywords in and not in. Lists can also be combined with the + operator and repeated with the * operator. And, just like strings, accessing an element from a list is done with indexing. As far as built-in python functions go, len(), min(), max(), and sum() all work as their names suggest with lists.

Task

Write a function listDouble that takes a list items, doubles it, finds the length of the resulting list and prints True or False whether the newly calculated length is a member of the list or not.

Sample Input 0

listDouble(['e', 78, 4, True, 10])

Sample Output 0

True
