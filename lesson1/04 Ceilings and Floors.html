Lesson 4: Ceilings and Floors

For our last bit with numbers, we'll explore a couple more built-in functions that python has to offer for us. The min() and max() functions can find us the smallest and largest values when given some data. This is also a good time to bring up arguments; back when we used print() we only gave it one value, like so:

print("Hello!")

The print function actually accepts multiple values, called arguments, which are separated by commas, for example:

print("Hello", "friends!")

For min() and max() to be of any use, they will of course need to accept multiple arguments. Go ahead and try it out in the exercise. We also have access to the abs() function, which gives us the absolute value of a given number.

Task

Write a function absMax that accepts three numbers x, y, and z, and prints the maximum of their absolute values.

Sample Input 0

absMax(2, -6, 3)

Sample Output 0

6
